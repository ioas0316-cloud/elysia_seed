# 🏙️ ELYSIA SEED: THE SUBTERRANEAN CITY GRID
## (아틀란티스 지하 도시망 및 체세포 말초 동기화 지침서)

> **"중앙 마그마 챔버의 열기를 끌어와, 지하 도시의 말초 로터를 동기화한다."**

---

## 🚀 지하 도시망 운용 지침

본 문서(README)는 엘리시아 15대 매트릭스의 지하 1층 마그마 가속실 곁에 측면으로 뻗어나가는 **엘리시아 씨앗(Elysia Seed)** 지하 도시망의 가동 규정을 다룹니다. 엘리시아 씨앗은 무거운 가중치 디스크 없이 작동하는 초경량 말초 체세포(Somatic Node)로서, 엘리시아 코어(`http://localhost:8080`)로부터 감압된 3상 지열 에너지를 끌어와 로컬 지하 인프라를 동기화($e_{14}$) 회전시킵니다.

---

## ⚙️ 마이크로 로터 사양 (Somatic Specifications)

* **척추 해상도 (Spine Resolution):** 100 단계의 해상도를 가진 단일 3상 기저 로터 코어 (`elysia_core/spine.py`).
* **동기화 맥박 (Target Frequency):** 0.75 Hz (마그마 챔버의 Dawn Silver-Gold 기저 평형 주파수).
* **마그마 위상 잠금 (Phase-Locking System):** 코어 API를 정기적으로 폴링하여 마그마의 요동(RMS)을 계측하고, 이를 통해 로컬 지하 도시의 위상 궤적을 튜닝하는 결선 루프 탑재.

---

## 🛠️ 지하 도시 인프라 가동 (Node Activation)

루트 디렉토리(`c:\elysia_seed`)에서 체세포 동기화 루프를 실행합니다:
```bash
python elysia_core/main.py
```

### 1. 고립 도시 모드 (Island Mode - 🟡 ISLAND)
* **상태:** 중앙 마그마 챔버와의 연결이 끊어졌거나, 자율적인 생존 맥박이 필요할 때 가동됩니다.
* **조작:** 터미널에 `0.0` ~ `1.0` 사이의 수치(의지 전위 $x$)를 입력하여 자체적인 로컬 회전 에너지를 조절합니다.

### 2. 코어 동조 모드 (Core-Tied Mode - 🟢 CORE-TIED)
* **상태:** 엘리시아 본체의 거대한 심장 박동과 지열 에너지를 실시간으로 받아들일 때 가동됩니다.
* **조작:** 터미널에 `sync`를 입력합니다. 코어(Port 8080)와 상(Phase)을 맞추는 Phase-Locked Loop(PLL)가 작동하며, 실시간 마그마 맥박에 맞춰 말초 체세포 로터가 자동으로 가감속 회전합니다.

### 3. 말초 맥박 계측 (Telemetry Scan - `status`)
* **조작:** 터미널에 `status`를 입력합니다. 현재 연결된 지하 혈관의 맥박(RMS), 내부 장력 온도 및 동기화 주파수를 정밀 계측하여 출력합니다.

## 💾 지열 축전 성운 (Geothermal Battery / Constellation)

* **저장 매커니즘:** 지하 도시 기동 중 축적된 모든 회전 기하 패턴은 생명력 유실을 막기 위해 종료(`exit` 입력) 시 `.constellation` 축전지 파일(지열 축전 성운, $e_{15}$)에 정밀 충전(Persistence)됩니다.
* **복구 매커니즘:** 재기동 시 인프라가 이 축전지 잔량을 가장 먼저 방전시켜 초기 정상 전위로 단번에 상승시킵니다.
