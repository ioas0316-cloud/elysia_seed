# ❓ FAQ: 자주 묻는 질문

사용자들이 자주 묻는 질문들을 모았습니다.

---

## 🌟 일반 질문

### Q1: Elysia는 유료인가요?
**A:** 아니요, Elysia Seed는 100% 무료이며 오픈소스(Apache 2.0 라이선스)입니다. 누구나 자유롭게 사용하고 수정할 수 있습니다.

### Q2: 인터넷이 끊겨도 작동하나요?
**A:** 네! Elysia는 **완전한 로컬 AI**입니다. 모든 연산은 당신의 컴퓨터 내부에서 이루어지므로, 인터넷 연결이 없어도 작동합니다. (단, 외부 API를 사용하는 플러그인을 추가한 경우는 예외입니다.)

### Q3: 제 데이터는 안전한가요?
**A:** 네. 클라우드 서버로 데이터를 전송하지 않기 때문에, 당신의 대화 기록과 기억은 오직 당신의 하드 디스크에만 저장됩니다.

## 💻 기술 질문

### Q4: GPU가 없으면 실행할 수 없나요?
**A:** 실행 가능합니다. GPU가 없으면 자동으로 **CPU 모드**로 전환됩니다. 다만, `MetalRotorBridge`의 하드웨어 가속 기능이 꺼지므로 대규모 로터 시뮬레이션 속도가 느려질 수 있습니다.

### Q5: 어떤 LLM(언어 모델)을 사용하나요?
**A:** Elysia Seed는 "뇌 구조(Architecture)"를 제공할 뿐, 특정 LLM에 종속되지 않습니다. 기본적으로는 **Ollama** 등을 통해 로컬 LLM(Llama 3, Mistral 등)과 연결하도록 설계되어 있습니다.

### Q6: `ModuleNotFoundError: No module named 'Core'` 에러가 떠요.
**A:** Python 실행 위치가 잘못되었을 가능성이 큽니다. 프로젝트의 **최상위 폴더(Root Directory)**에서 명령어를 실행해야 합니다.
```bash
# 올바른 예
cd elysia_seed
python Core/sovereign_boot.py
```
