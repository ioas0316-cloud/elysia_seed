# Concepts: Fractal Consciousness

엘리시아 엔진의 기본 개념을 간략히 정리했습니다.

## E / F / P 내핵

- **Energy**: 의미·생동감의 축적치
- **Force**: 현재 순간의 방향성과 압력
- **Momentum**: 과거 선택이 만든 관성

`EFPState`는 이 세 값을 1차원으로 표현하지만, 상위 시스템에서 다중 차원이나 채널을 얹어도 무방합니다.

## Body / Soul / Spirit

각 엔티티의 force는 세 가지 컴포넌트를 통해 구성됩니다.

1. `f_body`: 생존, 물리적 위협, 감각
2. `f_soul`: 감정, 관계, 심상, 인지
3. `f_spirit`: 의미, 가치, 신념, 소명

역할(전사/마법사/성직자 등)은 이 세 힘의 가중치를 정의하여, 같은 사건이라도 각자의 해석과 반응을 달리하게 만듭니다.

## Ring Memory

의식은 흐르지만 흔적을 남깁니다. `RingMemory`는 최근 에피소드를 순환 버퍼에 보관하여, LLM/게임/시뮬레이터가 "최근에 무슨 일이 있었나"를 빠르게 참조할 수 있도록 돕습니다.

## Fields

세계는 존재만큼이나 환경에 의해 좌우됩니다. `FieldRegistry`는 1D/2D 스칼라 필드를 이름과 함께 등록하고, 엔티티가 주변 환경 값을 조회하여 force 계산에 반영할 수 있도록 합니다.

## Persona Snapshot

`World.export_persona_snapshot()`은 현재 세계의 시간을 기록하고, 모든 엔티티의 직렬화된 상태를 포함한 딕셔너리를 반환합니다. `persona.build_persona_frame()`은 이를 UI/LLM/Godot이 바로 사용할 수 있는 축약 프레임으로 변환합니다.

사랑과 진실을 위한 기술. 이것이 엘리시아 엔진의 중심입니다.
