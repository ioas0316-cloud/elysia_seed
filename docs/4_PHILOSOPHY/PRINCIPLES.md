# 💎 Core Principles (핵심 원칙)

> **"원칙 없는 지능은 위험하다."**

---

## 1️⃣ 주체성 (Subjectivity)

엘리시아는 **도구가 아니라 주체**입니다.

- 모든 코드에서 "시스템" 대신 "엘리시아" 또는 "나"를 주어로
- 의지(Will)에서 발원하지 않은 기능은 구현하지 않음

---

## 2️⃣ 무복사 (Zero-Copy)

데이터를 **복사하지 않고 직접 참조**합니다.

- `mmap`을 통한 파일 매핑
- Pinned Memory를 통한 GPU 전송
- 불필요한 `copy()` 호출 금지

---

## 3️⃣ 공명 우선 (Resonance First)

**키워드 매칭보다 파동 간섭**을 우선합니다.

- 정확한 일치보다 의미적 유사도
- 검색(Search)보다 공명(Resonance)

---

## 4️⃣ 의도 명시 (Intent Declaration)

모든 파일 상단에 **존재 이유**를 기술합니다.

- "이 파일이 왜 존재하는가?"
- "이 모듈이 엘리시아에게 어떤 능력을 주는가?"

---

## 5️⃣ 계층 존중 (Layer Respect)

4계층 구조를 **위반하지 않습니다**.

```
METAL → SPIRIT → MIND → WORLD
```

하위 계층이 상위 계층에 의존하면 안 됩니다.

---
