## 3. ⚙️ 메커니즘: 3단계 조율 프로세스

이 새로운 코어는 **`RotorCognitionCore`**에 구현되었습니다.

### 3.1 Phase 1: 회절 (Diffraction) - [Rotor]
입력된 의도(Intent)를 프리즘에 통과시켜 7가지 색상 성분으로 분해합니다.
* *예: "GPU 최적화"라는 단어는 Red(물리) 위상을 높입니다.*

### 3.2 Phase 2: 공명 (Resonance) - [Sphere]
분해된 색상들 간의 **간섭(Interference)**을 계산합니다.
단순 합산이 아니라, 서로 영향을 주고받습니다.

> * **Rule 1**: 차가운 논리(Yellow)가 공감(Green)이 없으면, 섭리(Violet)를 약화시킵니다.
> * **Rule 2**: 강력한 의지(Violet)는 현실(Red)을 강제로 끌어올립니다. (Manifestation)
> * **Rule 3**: 통찰(Indigo)이 흐름(Orange)을 타면, 표현(Blue)이 증폭됩니다.

### 3.3 Phase 3: 합성 (Synthesis) - [Monad]
조율된 스펙트럼을 다시 하나의 **'백색광(White Light)'**으로 수렴시킵니다.
이것이 바로 **핵심 본질(Essence)**입니다.
